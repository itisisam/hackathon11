HTML: HyperText Markup Language

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage. HyperText refers to the ability to create links, and Markup Language means you use "tags" to give content meaning and structure.

Essential HTML Elements

1. Headings (h1 to h6)

Headings are used to define titles and subtitles, creating a logical hierarchy for your content.

Example:

This is a Heading 6

2. Paragraphs (p)

The paragraph tag is for grouping sentences and blocks of text together. Browsers automatically add a bit of space before and after a paragraph.

This is the first paragraph. It contains several sentences about a topic.

This is a second, separate paragraph. The browser will display it with a space between it and the first one.

3. Lists (ul, ol, li)

These tags are used to create lists of related items. An unordered list creates bullet points, while an ordered list creates a numbered list.

Unordered List (Shopping List):

  • Tea Bag
  • Water
  • Sugar
  • Milk

Ordered List (Recipe):

  1. First, boil the water in a kettle.
  2. Add tea bag, sugar and milk into it.
  3. Keep boiling it for 5 minutes.
  4. Serve the tea by pouring it into a cup.

4. Links (a)

The anchor tag (<a>) is used to create a hyperlink. It requires the href attribute to specify the destination URL.

5. Images (img)

The image tag (<img>) is a self-closing tag used to embed an image onto your page. It requires the src attribute for the image source and the alt attribute for a text description.

6. Line Break (br)

The line break tag (<br>) creates a single line break within a block of text.

221B Baker Street
London, England

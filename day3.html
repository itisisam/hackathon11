HTML Nested Lists & Tables

This document provides a summary of nested lists and tables in HTML, demonstrating their structure and use.

HTML Nested Lists

Nested lists are lists that are placed inside other lists to create a hierarchical structure for information. You can nest either an unordered list (<ul>) or an ordered list (<ol>) inside a list item (<li>).

Example of Nested Lists:

  • Fruits
    • Apple
    • Banana
  • Subjects
    1. Math
    2. Science

HTML Tables

Tables are used to display data in a grid format with rows and columns. They are ideal for presenting structured information like financial data or schedules.

The main tags used to create a table are:

  • <table>: The main container for the entire table.
  • <tr>: Stands for "table row." Each defines a single row.
  • <th>: Stands for "table header." Used for heading cells.
  • <td>: Stands for "table data." Used for regular data cells.

Example of a Table:

Month Savings
January $100
February $80
